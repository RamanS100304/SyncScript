# Realtime Code Editor

A Realtime Collaborative Code Editor built with:

- React.js (Frontend)
- Node.js + Express + Socket.IO (Backend)
- CodeMirror editor

Users join a room using a Room ID, and all code edits sync instantly across connected clients.
